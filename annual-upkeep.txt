1. Clear out the db from last season by deleting main.db
2. Update the json file at _constants/schedule-latest-UPDATEME.json and set the dates for each week (cutoff date is the date+time of the first game of each week, usually thursday night). Cutoff dates stop players from changing picks after a certain date/time. 
    start date is always tuesday night/wednesday morning after the nfl week completes (wednesday 2am)
3. Set the sleeperIds of A and B League in _constants/sleeper-leagues.json
3. Run 'npm start' to create the model and start server
4. Games are now ready
